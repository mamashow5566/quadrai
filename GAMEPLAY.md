# Quadra 遊戲規則與操作指南

## 基本操作

| 按鍵 | 功能 |
|------|------|
| 左右方向鍵 | 移動方塊 |
| 下方向鍵 | 軟降（加速落下） |
| 上方向鍵 | 旋轉方塊（順時針） |
| Space | 硬降（直接落底） |
| Pause/Break | 暫停 / 開始倒數 |
| Enter | 確認選單 |
| Esc | 返回 / 取消 |

## 遊戲目標

消除橫向填滿的行數來得分。方塊堆到畫面頂端即死亡。

## 攻擊規則（多人對戰）

消除行數越多、連續消除（Combo）越多次，送給對手的攻擊行數越多：

| 消除行數 | 攻擊行數 |
|---------|---------|
| 1 行（Single） | 0 |
| 2 行（Double） | 1 |
| 3 行（Triple） | 2 |
| 4 行（Tetris） | 4 |

**Combo 加成**（連續下放都有消行）：

| Combo | 額外攻擊 |
|-------|---------|
| 第 2 次 | +1 |
| 第 3 次 | +2 |
| 第 4 次以上 | +3 |

收到的攻擊行會從底部推上來，附帶一個隨機空洞。

## 遊戲模式

| 模式 | 說明 |
|------|------|
| **Free for All** | 標準死鬥，互送攻擊行 |
| **Survivor** | 最後存活隊伍獲勝，死亡後重生 |
| **Peace** | 無攻擊，純比分數 |
| **Blind** | 對手畫面隱藏，看不到對方狀態 |
| **Hot Potato** | 消除行數傳給對手，持炸彈者壓力最大 |

## 多人連線設定

### 1. 安裝 qserv 伺服器
啟動 `start.bat`（預設 port 3456）

### 2. Quadra 設定
- Options → Advanced → **Game server address**：伺服器 IP 或域名
- Options → Advanced → **Port**：遊戲託管 port（預設 27910）

### 3. 開 Host（公開遊戲）
- Multi Player → TCP/IP Internet → **Host a new game**
- **Public: Yes**（必須，否則遊戲不會刊登）
- 選擇遊戲模式與設定 → **Start Game**
- 如有 NAT，路由器需 forward port 27910

### 4. 加入別人的遊戲
- Multi Player → TCP/IP Internet → **Refresh**
- 看到遊戲後點選 → **Join**

### 5. 開始對戰
- Client 加入後，Host 按 **Pause/Break** 鍵開始倒數
- 倒數結束自動開始

## 計分系統

| 消除行數 | 分數 |
|---------|------|
| 1 行（Single） | 100 |
| 2 行（Double） | 250 |
| 3 行（Triple） | 500 |
| 4 行（Tetris） | 1000 |

分數會隨等級提升而加成（等級越高，同一消除行數得分越多）。

## 分數與排行榜

### 本地排行榜
每局結束後，若分數打進本地前 N 名，會自動儲存並記錄該局的完整錄影（demo）。

### 全球排行榜（World Highscores）

**存放資訊**：

| 欄位 | 說明 |
|------|------|
| `name` | 玩家名稱 |
| `score` | 最高分 |
| `lines` | 消除行數 |
| `level` | 到達等級 |
| `rec` | 完整遊戲錄影（可用 Quadra 播放） |

**更新方式**：

1. 打進**本地排行榜**的紀錄才具上傳資格
2. High Scores → **Sync** → 自動上傳本地最高分 + 下載全球榜
3. 伺服器保留全球前 **100 筆**最高分，超過自動清除
4. 點擊全球榜上的紀錄可**觀看該玩家的完整遊戲錄影**

**沒有本地紀錄時**：Sync 只會下載全球排行榜，不上傳。

## 常用指令（Host 聊天框輸入）

| 指令 | 功能 |
|------|------|
| `/allowstart 1` | 允許開始遊戲 |
| `/public 1` | 設為公開遊戲 |
| `/auto_restart 1` | 遊戲結束自動重開 |

## 常見問題

| 問題 | 解法 |
|------|------|
| 看不到遊戲列表 | 確認 Game server address 正確、qserv 有執行 |
| 遊戲暫停無法開始 | 按 **Pause/Break**（不是字母 P） |
| 對戰中無法送攻擊 | 一次消 2+ 行才送攻擊；消 1 行不送 |
| Client 連不上 Host | Host 路由器需 forward port 27910、防火牆允許 |
| 遊戲列表顯示內網 IP | NAT 後的 Host 會自動偵測公網 IP |
